# Validation data

Single-wheel testbed data digitized from published papers. Used **only** to
validate the evaluator; never used for training the surrogate.

## Target sources

- Ding et al. 2011 — Lunar rover single-wheel slip/sinkage experiments.
- Iizuka & Kubota 2011 — Grousered wheel experiments.
- Wong — datasets from *Theory of Ground Vehicles* (4th ed.).

Keep the raw digitized CSVs in `raw/` (git-ignored, re-downloadable from
the papers) and curated, checked-in versions at this level.
